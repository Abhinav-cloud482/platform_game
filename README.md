# platform_game

## Advanced Platformer Game (Pygame)

A simple yet feature-rich 2D platformer game built using **Pygame**. This project demonstrates core game development concepts such as player movement, gravity, collision detection, enemies, collectibles, and level progression.

##  Features

* Smooth player movement (left, right, jump)
* Gravity and platform collision physics
* Multiple levels with increasing difficulty
* Collectible coins with score tracking
* Moving enemies with patrol behavior
* Level progression system
* Win screen after completing all levels

## Tech Stack

* Python 3
* Pygame

## Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/platformer-game.git
   cd platformer-game
   ```

2. Install dependencies:

   ```bash
   pip install pygame
   ```

3. Run the game:

   ```bash
   python main.py
   ```

---

## Controls

| Key          | Action     |
| ------------ | ---------- |
| A            | Move Left  |
| D            | Move Right |
| Space        | Jump       |
| Close Window | Exit Game  |

## Game Mechanics

* **Gravity System** : Simulates realistic falling and jumping.
* **Collision Detection** : Prevents the player from falling through platforms.
* **Enemy AI** : Enemies move back and forth within a defined range.
* **Score System** : Collect coins to increase your score.
* **Level System** : Progress through levels by collecting all coins.
